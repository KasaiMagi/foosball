API Endpoints

/api/users.json

A list of all users on the platform with the following attributes:

  • id
  • name
  • wins
  • losses
/api/matches.json

A list of all matches with the following attributes:

  • id
  • teamA - containing two user IDs
  • teamB - containing two user IDs
  • scoreA - teamA's score
  • scoreB - teamB's score
